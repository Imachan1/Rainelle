# Project Roadmap

## Phase 1: Foundation

- Laravel backend
- Sanctum authentication
- Vue frontend
- Mood entry CRUD foundation
- Basic docs

## Phase 2: Mood Tracking

- Improve mood entry forms
- Add edit/delete UI

- Add filtering by date

## Phase 3: Weather Integration

- Add weather provider configuration
- Save weather snapshot with mood entries
- Show weather context in entry details

## Phase 4: Insights

- Prepare chart-friendly API data
- Add D3 mood trends
- Add mood/weather comparison charts

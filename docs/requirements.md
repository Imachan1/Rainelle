# Requirements

## Goal

Rainelle helps users track daily mood entries alongside weather context.

## Core Features

- User registration and login
- Daily mood entries
- Mood intensity from 1 to 10
- Optional notes
- Optional emotions list
- Weather data connection later
- Mood/weather insights later

## Out of Scope For Initial Setup

- D3 charts
- Advanced statistics
- AI features
- Social features
- Mobile app
